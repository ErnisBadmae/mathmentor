# Единый рабочий цикл с Codex

`ACTIVE_TASK.md` — общая карточка текущей задачи для Claude и Codex. `SESSION_LOG.md` — общая история сессий (верхняя запись — самая свежая).

В начале сессии: прочитай `ACTIVE_TASK.md` и верхнюю запись `SESSION_LOG.md`, прежде чем лезть в код.

Не веди отдельный параллельный task log — это единственный источник состояния для обоих агентов.

В конце сессии: обнови `ACTIVE_TASK.md` (цель/статус/решения/следующие шаги) и добавь короткую запись в `SESSION_LOG.md` (задача и статус, 3-6 пунктов по сути, проверка, открытый хвост).
